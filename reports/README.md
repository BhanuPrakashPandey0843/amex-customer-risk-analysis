# reports/

## Purpose
This directory contains all the deliverables generated by the analysis pipeline: executive reports, structured insights, prioritized recommendations, a risk register, publication-quality charts, and an Excel workbook.

## Directory Structure
```
reports/
├── charts/                              # 9 publication-quality visualizations (PNG)
├── executive_presentation.pdf           # Auto-generated PDF (data validation deck)
├── CFPB_Executive_Analytics.xlsx        # 8-sheet Excel workbook with full analysis
└── [30+ Markdown reports]               # Structured documentation
```

## Regeneration
All files in this directory (except the designed presentation in `data/ppt/`) can be regenerated by running:
```bash
python src/run_pipeline.py
```

## Important Notes
- Every number in every report comes from `data/processed/eda_metrics.json`
- No file in this directory should be edited manually — re-run the pipeline instead
- The Excel workbook has 8 sheets, one for each major section of analysis
